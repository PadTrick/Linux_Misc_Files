After Installing Archlinux:


#Enable Color, Pacman progress and lib32 support.
Edit /etc/pacman.conf and change "#Color" to "Color", add "ILoveCandy" below it. Change "#ParallelDownloads = 5" to "ParallelDownloads = 5". Scroll to the bottom and find
"#[multilib]
#Include = /etc/pacman.d/mirrorlist"
and remove the "#" at the front to look like this
[multilib]
Include = /etc/pacman.d/mirrorlist

after that save it.


#Disable debug package build.
Now edit /etc/makepkg.conf and find the lines
DEBUG_CFLAGS="-g"
DEBUG_CXXFLAGS="$DEBUG_CFLAGS"
and put a # infront
#DEBUG_CFLAGS="-g"
#DEBUG_CXXFLAGS="$DEBUG_CFLAGS"
then find the line "OPTIONS=(strip docs !libtool !staticlibs emptydirs zipman purge debug lto)"
and put a ! infront of debug
OPTIONS=(strip docs !libtool !staticlibs emptydirs zipman purge !debug lto)

after that save it.


#Increase shadercache size for your GPU
Edit /etc/environment depending on your GPU add one of the following blocks.

#This block is for NVIDIA
# The following commands are for NVIDIA GPUs like my old GTX 1070Ti etc.
__GL_SHADER_DISK_CACHE=1
__GL_SHADER_DISK_CACHE_SKIP_CLEANUP=1
__GL_SHADER_DISK_CACHE_SIZE=100000000000
#__GL_SHADER_DISK_CACHE_PATH=/var/cache/shaders

# Enable threaded optomisation for OpenGL.
__GL_THREADED_OPTIMIZATION=1


#This Block is for AMD & Intel GPUs
# The following commands are for Mesa (AMD/Intel)
MESA_SHADER_CACHE_DISABLE=false
MESA_SHADER_CACHE_MAX_SIZE=100G

after that save it.
